# Raspberry Pi Pico Config

Run Thonny on Raspberry Pi 4, with a USB connection to the Raspberry Pi Pico (standard variant). In Thonny, select "MicroPython (Raspberry Pi Pico)" as the interpreter, then run the file `src/rpi_pico/pico_config.py` to configure the Raspberry Pi Pico. This will set up the necessary configurations for the Pico to communicate with the Raspberry Pi 4.
